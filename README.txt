Install

    Archive contains full source codes and two binaries, Japanese version
    (hambiffj.exe) and Engish version (hambiffe.exe).   If you use
    binary only, then copy

        hambiffj.exe
	    or
	hambiffe.exe

    to a directory in PATH and invoke from command line, or place it in
    some directory and create program object for it.  For more details,
    see document 'hambiff.txt'.

    This program was written for EMX/GCC.  You need EMX runtime (0.9c
    or later) to run this program.
